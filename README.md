# Monitoria – Econometria I (Graduação) – 2026

Este repositório reúne os materiais desenvolvidos para a monitoria da disciplina **Econometria I (Graduação - FEA/USP)** (EAE1221) no ano de 2026.

O objetivo é fornecer suporte complementar aos alunos por meio de:
- Listas de exercícios
- Soluções comentadas
- Introdução ao software R
- Scripts de estimação e simulação
- Exemplos aplicados aos modelos econométricos estudados na disciplina

## Estrutura do Repositório

- [Listas](./Listas/) – Listas aplicadas ao longo do semestre  
- `R_Intro/` – Materiais introdutórios de R  

## Bibliografia Principal

Tenho buscado exercícios em:

- WOOLDRIDGE, Jeffrey M. *Introdução à Econometria: Uma Abordagem Moderna*. 4ª ed. São Paulo: Cengage Learning Brasil, 2023.
